# 沖繩親子旅行行程規劃書

這是一個詳細的沖繩親子旅行行程規劃網站，包含了 2025 年 6 月 11-16 日的完整六天五夜行程。

## 功能特色

- 詳細的每日行程安排
- 交通資訊（含租車建議）
- 美食推薦
- 購物清單
- 行前準備事項
- 緊急聯絡資訊

## 使用技術

- HTML5
- CSS3
- JavaScript
- 響應式設計

## 如何使用

1. 直接訪問 GitHub Pages 網站：[https://kagexx.github.io/okinawa-trip/](https://kagexx.github.io/okinawa-trip/)
2. 或下載原始碼後在本地瀏覽器中開啟 index.html
